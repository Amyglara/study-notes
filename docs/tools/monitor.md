# 行情监控工具

一套好的监控体系能让你第一时间捕捉机会，也能在风险来临前提前预警。

## K 线与技术分析

### TradingView — 行情分析标准工具

```
网址：tradingview.com
费用：免费够用，Pro 版 $15/月（多图表同时看）
```

**必会操作**：

1. **添加加密货币**：搜索 `BTCUSDT.P`（P 结尾 = 永续合约，价格最准）
2. **切换时间周期**：1D / 4H / 1H / 15M
3. **添加指标**：
   - 顶部搜索栏输入 "RSI" / "MACD" / "EMA"
   - 推荐添加：EMA(9) + EMA(20) + EMA(200)
4. **画支撑阻力线**：左侧工具栏 → 水平线工具
5. **设置价格提醒**：右键 K 线图 → "Add Alert"

**TradingView 提醒设置**：
```
条件：RSI(14) crossing above 30
时间框架：4 小时
通知方式：浏览器推送 / 邮件 / 手机 App
```

### CoinGecko / CoinMarketCap

```
CoinGecko：coingecko.com
CMC：coinmarketcap.com
```

**每日必看**：
- 24h 涨跌幅排行（什么在爆拉？什么在暴跌？）
- 新上线代币（快速过滤新热点）
- 全球市值占比（BTC 主导率）

**BTC 主导率（BTC Dominance）解读**：
```
BTC 主导率上升 → 资金流向 BTC，避险情绪浓
BTC 主导率下降 → 资金扩散到山寨，山寨季来临迹象

历史规律：
  BTC 主导率 > 55% 且开始下降 → 可关注山寨机会
  BTC 主导率快速上升 → 警惕山寨大跌
```

### Coinglass — 合约数据专项

```
网址：coinglass.com
费用：免费
```

**核心功能**：

| 功能 | 用途 |
|------|------|
| **清算热力图** | 哪个价格区域有大量多/空单待清算 |
| **资金费率** | 正高 = 多头拥挤（做空机会）；负值 = 空头拥挤 |
| **未平仓合约量** | 总 OI 骤降 = 大规模清算发生 |
| **多空比** | 交易所整体多空比例 |
| **爆仓数据** | 近 24h 多空爆仓金额 |

**清算热力图使用**：
```
价格下方有大量多单 → 下跌可能触发大规模多单清算 → 下跌加速
价格上方有大量空单 → 上涨可能触发空单清算 → 上涨加速（空头回补）

当你看到价格上方密集空单时：
  这是"轧空"（Short Squeeze）的潜在燃料
```

## 提醒系统搭建

### 方案一：TradingView 价格提醒

```
步骤：
1. TradingView 添加代币图表
2. 右键 → "Add Alert to chart"
3. 设置条件：价格突破/跌破某个关键位
4. 通知方式：勾选"Send email" + "Notify on app"
```

### 方案二：交易所 App 提醒

大多数交易所 App（币安/OKX）支持：
- 价格到价提醒
- 涨跌幅提醒（24h 涨跌超过 X%）

### 方案三：麒麟会情报中心

麒麟会每日推送：
- BTC/ETH/主流币价格快照
- 恐慌贪婪指数解读
- 市场信号（多/空）
- 重要新闻摘要

→ [查看早晚报](https://zx.bitdong.xyz)

## 综合监控工作流

```
每日例行（10分钟）：
  1. 看麒麟会早报 → 了解昨晚发生了什么
  2. 看恐慌贪婪指数 → 市场情绪定调
  3. TradingView 刷一遍 BTC 日线 → 趋势是否变化
  4. Coinglass 看资金费率 → 合约情绪是否极端
  5. CoinGecko 涨跌榜 → 今天什么在动

每周复盘（30分钟）：
  1. 回顾本周信号执行情况
  2. 检查持仓健康因子（如有 DeFi）
  3. 用 Glassnode 看宏观链上指标
  4. 更新交易记录
```
