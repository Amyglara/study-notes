# 区块链红宝书 📕

> **最接地气的币圈词典**

## 前言

愿为币圈人，点亮一盏灯。

币圈人都知道一句话：**你永远赚不到你认知外的钱，凭运气赚的钱也都会凭本事再赔出去。**

近期跟一些做投研的朋友聊天，他们都有一种感觉：睡一觉起来之后，世界就变了。在这个世界上每天都有新的项目、新的玩法、新的概念产生，这些做投研的科学家们都有点研究不过来了。

所以在这里出现了两个极端：一方面是日新月异的区块链世界，一方面是认知几乎为零的币圈新人。一方面是每天都赚麻了的科学家，一方面是被割的找不到头脑的韭菜。

良心哥认为需要有人来为行业做点事情，去缩短、填平这中间的认知差。不求能带人在这过程中赚多少钱，只求能提升韭菜们的短板和认知。这就是《区块链红宝书》的由来。

## 本书特点

1. **通俗易懂的解释和案例**：对专业术语有通俗易懂的解释和相应的案例展示。
2. **关联词汇拓展**：提供同义词、近义词、反义词的解释，延伸相关概念。
3. **黑话和梗的定义**：对圈内黑话和梗进行专业释义。
