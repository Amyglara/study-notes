# 第六章：会管理仓位才能盈利

## 本章学习要点

::: tip 核心要义
1. **全仓是注定失败的**，从未见过上来就全军突击的战胜案例。
2. 把资金分为几部分，各自赋予使命和进退指令，你的投资会变得有序。
3. **334战法**是攻守兼备的战法。
4. 趋势投资本质是左侧投资。
:::

---

## 一. 全仓必亏

《孙子兵法·兵势篇》中孙子说：

> 善于调动敌人的将领，只要摆出一些样子，敌人就会傻傻跟来。给他们一些诱饵，他们就会来夺取。等敌人被利诱来了，就可以埋伏精兵强将收拾敌人。

是不是有点熟悉？

> 币价横盘，你就撤资换币；币价一涨，你就买币追高；币价一跌，你就卖币逃窜。

**庄家让你向左，你就向左；庄家让你向右，你就向右。**

为什么会这样？**因为你选择了全仓。**

全仓的代价：
- 押上了全部，意味着你没有退路
- 庄家把币价拉伸10%，你感觉受益颇丰，想继续追击
- 刚刚追上，庄家伏兵四起打压币价跌15%，你犹豫不决
- 继续砸盘跌40%，你焦虑崩溃卖币投降
- 然后庄家一路高歌猛进，币价拉伸50%，你完美错过后追入继续被套

::: danger 全仓的本质
全仓，意味着你全身赤裸暴露在庄家面前，进无可进，退无可退。

参考长平之战的赵括，街亭之战的马谡，都是全军被困，失去所有机动性。
:::

---

## 二. 分兵战法

孙子说：

> **三军之众，可使必受敌而无败者，奇正是也。凡战者，以正合，以奇胜。**

只要你不押上全部，就进退有据。哪怕币价回调，本金浮亏，你也可以镇定下来分析是庄家震仓还是真的下跌：

- 如果是震仓 → 毫不犹豫继续加仓，来个反伏击
- 如果是下跌 → 亏损并不大，还可以保存实力再战

克劳塞维茨在《战争论》也谈道：

> "双方留下的预备队的对比，往往是最后决定胜负的主要根据。"

**我们的钱就是我们的兵，买成币以后就意味被推上了战场。所以，在上阵前，好好编队，好好分工，明确进退的依据和信号。**

## 三. 334战法

所谓334，指：
- **3成** → 左仓
- **3成** → 右仓
- **4成** → 决战仓

### 左仓（30%）— 步兵

**负责从山底一直打到山顶，完成试探虚实、肉搏等所有工作。**

从币价底部开始建仓，一步不退的与庄家拼杀。这30%买入就不要乱动了，直到币价到达山顶，赚取约100%利润。

### 右仓（30%）— 骑兵

**每当看见局部战机，就冲进去，赚了就走，头也不回。**

币价在一定时期都是沿着通道有规律运行。每当币价到了箱底，又会反弹向上，到了箱顶，又会反弹向下。在4个波段中来去自如，每个波段收益10%左右。

::: warning 右仓铁律
1. 只要拿到利润就要撤退
2. 跌破建仓价格就要卖出
3. 右仓永远不能被套
4. 右仓只挣大概率的钱
:::

### 决战仓（40%）— 嫡系精锐

**不到趋势明朗、全军突击时不动。**

等庄家所有震仓花样玩完，准备暴力拉伸时，全部投入进去，斩将夺旗。

## 四. 334战法实战演示

以预计投入100,000元参与某币为例：

| 仓位 | 金额 | 操作说明 |
| --- | --- | --- |
| 左仓 | 30,000元 | 底部分批买入，持有到顶部 |
| 右仓 | 30,000元 | 在通道内4次波段操作，每次10%+ |
| 决战仓 | 40,000元 | 震仓结束，暴力拉升时全仓冲入 |

**结果预估：**
- 左仓收益：30,000元（100%利润）
- 右仓收益：12,000元（4次×10%）
- 决战仓收益：20,000元（50%暴涨）

**10万本金，累计盈利87,000元，年化收益率极高。**

## 六. 趋势投资 vs 334战法

趋势投资本质是**左仓投资**：

::: info 两个关键前提
1. **闲钱投资**：趋势投资的资金是在投资周期中对你生活无影响的闲钱。
2. **估值背离**：只有在估值和价值背离很严重的时候，才有足够势能去修正。如果弹簧拉的不够开，就不会回去。
:::

::: warning 关于全仓某币
当你听到某某人全仓某币，请先判断他所谓的"全仓"是全身家当，还是规划用于币市投资的钱。

- 如果是前者 → 这是傻子，即使挣了钱也不值得学习，凭运气赚的钱，也会凭本事赔出去。
- 趋势投资本质是左侧投资，并不是把全部的钱全部盲目买入。
:::

::: tip 最终总结
- **认知能力**很重要，但**执行力**更重要。
- 懂得许多大道理，依然过不好这一生，原因在于只有道理没有底层的操作系统。
- **334战法**就是你的操作系统：左仓守阵地，右仓打游击，决战仓一击必杀。
:::
