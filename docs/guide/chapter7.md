## 问题一：现在开始玩比特币到底晚不晚？

*编辑于2024-02-18*

### 首先定义玩儿法

BTC的玩儿法特别多，不同玩儿法对应的风险和投资回报以及难度不一样：

| 玩儿法 | 难度 | 建议 |
| --- | --- | --- |
| 挖矿 / ICO / IEO | ★★★★★ | 一般人不考虑，小虾米只有被玩儿的份 |
| 定投 / 现货 | ★★☆☆☆ | 适合多数人，先把K线看懂 |
| 合约 / 期权 | ★★★★☆ | K线都看不懂的小白，基本上是火葬场 |

::: danger 警告
对于K线都看不懂的小白，合约和期权基本上没你们什么事。基本上就是火葬场，抱着钱往里面跳。
:::

### 从现货的角度看BTC

**先说结论：**
1. 本回答发表期间不是最好的投资时机；
2. 本回答发表期间可以采取**保守方法**投资。

**最佳投资期的逻辑：你应该在低位介入。** 底部都是有特征的，以下3种常用判断底部的方法：

#### 方法一：九神指数 AHR999

| 指数区间 | 含义 | 操作建议 |
| --- | --- | --- |
| < 0.45 | BTC被低估 | **抄底** |
| 0.45 ~ 1.2 | BTC正常估值区域 | **买买买** |
| > 1.2 | BTC被高估 | 根据情况决定是否逃顶 |

> 事实证明，如果九神指数在0.45以下你抄底了，收益都不会低于100%。

#### 方法二：5日线 KDJ 呈 W

- KDJ W形金叉右侧对应的那根K线，基本上都是最佳介入时机
- 最大振幅不超过10%，风险极低
- 介入后基本收益 30% 起，最高到 300%

#### 方法三：周线 MACD 底背离

- 底背离后，从抛压减弱那一根K开始介入
- 收益率貌似都没低于 **300%**

## 问题二：做交易，理智重要吗？

*编辑于2022-02-08*

**答：理智的前提是有理智的基础。**

没有技术分析基础，没有制定交易策略的能力，没有养成良好的止盈止损纪律，凭什么说自己可以保持理智？

::: warning 认清本质
**脱离了基础谈保持理智，都是空话。**

就像成功学一样——只谈成功者的心态，不谈成功的路径。只会让人信心满满上战场，一声枪响当逃兵。
:::

### 真正的技术分析三要素

很多人学习技术分析只是学习预判走势，但这只是指导交易的一小部分。真正的技术分析应当包括：

1. **预判走势** — 判断方向
2. **预判止盈位** — 至少涨到哪里？跌到哪里？
3. **找到最佳开仓点** — 开单就产生正盈利的点位

如果一个人只给你分析走势，基本上技术不过关，多半是嘴炮。

### 理想的交易状态

> 开单后都是不看的，设置自动平仓点位后就去玩儿了。中午开单，然后去打个游戏做个SPA，回来以后一看，嗯，今天玩儿的钱都赚回来了。

一个好的交易系统一定是：
- 能够预判涨跌
- 能够预判必然止盈位
- 开单无浮亏

否则，这就不是一个好的系统，无法保持所谓的理性。

## 问题三：比特币的暴跌可以预测吗？

*编辑于2020-03-22*

**可以预测。** 而且不是事后诸葛亮，而是提前使用的，每一次暴跌前的聊天记录都是有的。

### 一. 如何逃顶

#### 感觉式逃顶（简单）

记住：**只要天天涨，就是暴跌之日。**

当你突然有一天发现连涨了几天了，当你觉得"我怎么这么牛逼，一天可以赚这么多钱"的时候，就该离场了。

#### 傻瓜式逃顶（实用）

把 **RSI 指标**拿出来，参数改为 (7, 14)，**用周线看**：

- 如果周线RSI死叉 → 熊市无疑，无条件清仓
- 2020年3月1日发出警示"熊市无疑，无条件清仓" → 2周后，币圈迎来史诗级大暴跌

#### 高手逃顶（复杂）

需要综合运用：**波浪理论、斐波那契、趋势线、图形、MACD、RSI** 等。

逃顶步骤：
1. 出现连续上涨，查看日线RSI是否超过90（超买区）
2. 用**斐波那契扩展**算出相对顶部位置
3. 等MACD走坏了撤退
4. 如果跌了，观察有没有跌破趋势线或重要均线（55、120）

> 2021年超级大牛市，在62000全部清仓，后续一年一直跌到15476。

### 二. 如何逃熊市阶段大跌

**方法：预判顶部 + 指标验证**

1. **预判顶部**：借助斐波那契回调、抛压线找到阻力位，多重验证（MACD、筹码分布图、布林、MA）
2. **用指标验证**：到了阻力位，1小时MACD会持续顶背离，出现假金叉

**持仓量的变化是重要信号：**
- 第一阶段：持仓量正常，在9000美金有阻力
- 第二阶段：持仓量大幅增加，但币价还是在9000美金 → 说明新进来的是多头（不是空头）
- 第三阶段：假突破 + 持仓量缩减 → 说明多头跑路了

## 问题四：BTC破位大跌，新一轮抄底机会来了吗？

*编辑于2019-10-20*

回答三个问题：**这是底部吗？值得抄底吗？如何抄底？**

### 这是底部么？

截至2019年10月20日，BTC币价跌至7954美金，理论上并不是最低点：

1. BTC的币价有一根长期支撑线，近期底部约在 **6000美金** 左右
2. BTC挖矿成本会随时间越来越高，形成成本支撑

### 历史规律参考

2013年减半行情后的规律：
1. **筑底大反弹**（+247%）
2. **减半行情前**一轮拉升（+137%）
3. **减半后**开启大牛市

如果规律适用，未来目标：
- 中期目标：14,000 ~ 21,000 美金
- 长期目标：100,000 美金（2022年预判）

### 怎么抄底（以10万元为例）

| 时机 | 操作 | 金额 |
| --- | --- | --- |
| 当下 | 买入底仓 | 10,000元 |
| 明年3月前 | 分批买进 | 90,000元 |
| 减半前1周 | 卖出8成 | 获利离场 |
| 减半后跌回支撑线 | 全仓买回 | 再次布局 |

> 这样折腾2年，10万变成30万其实是比较容易的事情。

## 问题五：拿山寨币亏了100多万，怎么回本？

*编辑于2019-08-18*

**核心结论：大多数的山寨币的币价大概率不会回到前高。**

### 一. 持续打压币价让庄家挣更多钱

| 轮次 | 买入价（美金） | 卖出价（美金） | 利润率 |
| :--- | :--- | :--- | :--- |
| 第一轮 | 100 | 500 | 500% |
| 第二轮 | 10 | 50 | 500% |
| 第三轮 | 1 | 5 | 500% |
| 第四轮 | 0.1 | 0.5 | 500% |

**【倍数比币价的绝对值更重要】**

当你明白收益率和币价没有直接关系，你才会惊醒一个现实：**大部分山寨币币价不会再回到前高了。**

- EOS 2018年：从3.8美金涨到23美金，庄家利润率 600%
- EOS 2019年：从1.5美金涨到8.6美金，庄家利润率 570%

币价虽然没有回到前高，但庄家的利润率是一样的。

### 二. 币圈规则允许持续打压

| 原因 | 说明 |
| --- | --- |
| 币价可以无限小 | 比特币可细分到小数点后8位，山寨币兑BTC可到0.00000001 |
| 币币无监管 | 无"操纵证券罪"紧箍咒，项目方可随意抛售 |
| 上市成本极低 | 给交易所百万千万即可，不珍惜 |
| 项目方没追求 | 赚几百几千万跑路即可，换下一个继续 |

### 回本的正确路径

::: info 务实建议
死拿山寨币大概率终身抱憾。

正确做法：
1. 接受现实，不要死等回到前高
2. 在合适的时机止损，保留本金
3. 学习第四章的庄家四步法，重新找低位介入机会
4. 把注意力从"自己的亏损"转移到"庄家是否挣到钱了"
:::

::: danger 最终警示
**币价不重要，重要的是倍数，越跌庄家越挣钱。**

大部分山寨币的币价都在往下走，走着走着就爆发一次，且永远回不了前高。
:::
