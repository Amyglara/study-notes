# 第一章：投资的刚需是避险

## 本章学习要点

::: tip 核心要义
1. **无避险，不投资。** 先学会不亏，再学会赚钱。
2. 高手之所以达成常人办不到的事情，是因为他们所有的注意力都放在避险，以至于他们能完美的完成所有动作。
3. 只看见利润，看不见风险的人，最后都将黯然离场。
:::

---

第一篇，我们从避险说起，这是一切投资的逻辑起点。

币圈有两个被用烂的案例，不管是什么项目，都会祭出来，仿佛出手就能秒杀一切。

**案例一：李笑来**

大家熟知的中国比特币首富，13年几百块入手大量比特币，到现在不知道翻了多少倍。大家都津津乐道他的名言："握住，别丢。"于是，所有项目都声称自己有价值，希望大家握到地老天荒。

**案例二：比特币披萨**

相传2010年5月22日，有个程序员用10000个比特币换了2个披萨，被人们传为笑谈。币圈更是把每年的5月22日定义为"比特币披萨日"(Bitcoin Pizza Day)。这个案例侧面印证了拿不住的后果。

这两个案例一出手，真真能够倾倒无数人。拿住，可能千倍、万倍的回报；拿不住，可能错失千倍、万倍的良机。于是，很多人被割了韭菜。因为他们犯了一个本质性的错误，忘记了巴菲特的名言——**保住本金，保住本金。**

---

## 投资的刚需是避险

**"无避险，不投资。"** 请记住这句名言。

这句话并不是 Bit东 说的，恰好是用来镇楼的李笑来老师说的。这里为李笑来老师平个反，很多人亏钱后怪李笑来让他们拿住，这只是因为你在断章取义李老师的话。

如果你读过或者听过李笑来的《通往财富自由之路》，可以看看他在"投资的刚需是避险"这篇文章。李笑来指出，所有的高手在完成所谓高难度的动作时，并非富贵险中求、孤注一掷。而是把注意力都放在如何避险上。

他在文中指出，他敢于买比特币的前提是这笔钱来自于刚刚卖掉大涨美股的收益。因此，他之所以能镇定自若的持有，是因为他并没有花本金投资。另外，他的书、他的工作能持续的收入，让他不会因为没有这笔钱而为生活发愁。后来，比特币无论怎么大跌他都不卖，只是因为离他的成本价还很低。

> 我价格很低的时候买入了大量的比特币，后来大涨也没卖，再后来大跌也没卖……于是就有人说了："笑来，你真大胆！"，也会有人评价："哎，做大事的人就要敢于去承担极大的风险……"真的吗？
>
> 真的像他们看到的、想到的那样吗？"价格很低"是指相对当前的价格。我买的第一批两千一百个比特币均价六美元（2011年3月，现在七百多美元（2016年12月）；另，比特币曾经涨到一千多美元，又经过两三次"腰斩"……其实那个时候人们同样认为已经太贵了，赚不到钱了……可实际上于我来说，我在这样的时候购买，并不是冒险，因为：如果比特币是对的，它一定不止这个价格。如果比特币是错的，即使崩盘出局，我只不过是把最初花费四千六百元人民币的股票，在其已经价值十几万美元的时候卖掉，之后再买入比特币，所以，即便都化为乌有，我也没什么不能承受的。
>
> 我从十几美元一路下跌的时候开始反复买入，最后均价一美元左右，直到预算花完，实在没钱再买了——这个过程中，从我的角度，我并没有冒险。最初买入的时候，我花的钱只是我一大笔利润的一小部分。后来价格大幅度涨跌时，离我的成本也早得很。

## 核心原则

::: warning 请牢记
**投资的本质是避险，是出本金操作。** 这句话，是一句真理，请记住。以后，有任何人，用任何消息，许任何愿，你也要牢记，不要轻信。
:::

很多人动辄谈项目前景、团队优势、资方背景、模式牛逼。须知：

- ETH 作为万币之首，也从 1422 美金跌落到 81.93 美金，**跌幅超过 1735%**
- 被吹捧为区块链3.0的 EOS，也从 23.08 美金跌落到了 1.52 美金，**跌幅超过 1518%**

那么，你能够忍受你的本金跌幅如此之大的后果吗？如果不能，请在投资前，思考清楚，如何避险。否则，入币圈就是来送钱的，绝无例外。

## 孙子兵法的启示

大家可以读读《华杉讲透孙子兵法》，书中观点和这些投资大咖不谋而合：

- 孙子上来就说，用兵是国家生死存亡的大事，不可不深思熟虑。
- 开战前就要计算胜率，多算胜，少算败，何况算都不算。
- 不要先想着如何战胜敌人，要先想着如何保全自己，只有立于不败之地，才能捕捉战机消灭敌人。

**这些话用在投资上，一回事：**
- 不考虑胜算的投资是错误的
- 不计算投资带来的影响是错误的  
- 不考虑避险是错误的

在币圈，有很多独眼龙，他们的眼里和心里只看得见"利润"，看不到"风险"。每个人都揣着财务自由的念想来到币圈，最后被割得惨不忍睹，仓皇逃窜。

::: danger 警示
**只看见利润，看不见风险的人，最后都将黯然离场。**

愿你入币圈后不要陷入魔障。
:::
