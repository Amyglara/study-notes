# 第四章：学会躺着挣钱

## 本章学习要点

::: tip 核心要义
1. **炒币最帅的姿势是躺着赚，让庄家为你打工。**
2. 阴跌吸筹 → 整理降本 → 跌破拉涨幅 → 假拉震仓，四个步骤完成后庄家才会拉伸。
3. **不要臆测底部。**
4. 重视周期，顺势而为。
:::

---

## 躺着挣钱意味着什么

2019年初，如果你省吃俭用买上10个BTC（币价3000美金），到了2021年3月你会发现，妈妈呀，我居然挣了57万美金（币价60000美金），折合人民币接近400万。

相同本金（30000美金）：
- 买 **ETH** → 赚72万美金，折合人民币近500万
- 买 **BNB** → 赚177万美金，折合人民币近1200万

所以，进入币圈，第一件事是**学会躺着赚钱**，再学习那些高阶的挣钱方式。事实上，躺着挣钱的概率和利润往往是最高的，且最不耗费精力。你可以一边做自己喜欢的事情、陪伴着家人、专心从事本职工作，然后一边挣大钱，何乐而不为。

---

## 庄家四步法

每个经历过大行情的币，都经历了五个共同阶段：

```
阴跌吸筹 → 整理降本 → 跌破增幅 → 震仓控盘 → 🚀 暴力拉升
```

当你决定投资某币前，**先观察他到哪个阶段了**。如果完成了前四步，就可以慢慢介入了，终有一天他们会大爆发，涨到你不认识自己。

---

## 第一步：阴跌吸筹

**目的：庄家低价拿筹码。**

炒币的本质就是做生意，币价便宜的时候买入，币价贵的时候卖出，赚取利润（价差）。

- 庄家手上没筹码的时候 → 打压币价买币
- 庄家手上筹码多的时候 → 拉升币价卖币

当庄家拉高币价把币卖给韭菜后，庄家手里只有钱没有币。此时，庄家需要让币价阴跌，逼迫韭菜交割筹码。

---

## 第二步：整理降本

**目的：拉低持币成本。**

读者有心请把各种币的K线研究一番，几乎每个币都有等边三角、直角三角、箱体、旗型等整理图形。**整理动作越大，时间越长，后续币价的涨幅就越高。**

庄家的收益 = 未来拉伸币价后的顶部价 - 成本价

所以，庄家必须通过整理的手段拉低持币成本。

**对倒如何降成本（举例）：**
1. 币价1元时买入100个币
2. 拉到2元时卖掉50个币，赚取100元
3. 打压币价到0.5元，用100元买入200个币

这样一轮回：
- 100元成本 → 持有300个币
- 持币成本从1元/个 → 降低到0.33元/个

::: info 经验规律
如果你看到一只币，币价在一段期间上上下下，你就要关注了。但凡有整理的币，未来都会拉伸，因为主力不会吃饱了没事干这么折腾自己。
:::

---

## 第三步：跌破拉涨幅

**目的：增加利润空间。**

庄家币拿够了，成本足够低了就该拉了吧？**不**，利润空间不够大，而且未来拉伸时还要给韭菜解套。

所以庄家在准备好了以后会向下跌破，去争取足够无成本的涨幅空间。

**关键结论：大部分的币价不会再回到前高了。**

庄家的逻辑：
- 从10元拉到50元，纯赚500%
- 拉到100就有风险了，庄家要真金白银给前高韭菜解套
- 拉回前高可能亏，不拉回前高一定赚

---

## 第四步：震仓控盘

**目的：高度控盘，清洗车上的韭菜。**

拉伸前必须高度控盘，否则拉不上去。原因很简单：如果庄家暴力拉伸币价，韭菜在车上就会一直卖，导致币价拉的很艰难，庄家还要真金白银接着韭菜抛售的币。

**震仓手法：**
1. **假动作**：币价突然上涨很多然后跌破前低（术语：抛压测试）
2. **消息震仓**：随便抛出一个消息让韭菜觉得世界末日来了，赶紧把币抛售

### 📌 BSV 经典案例

2019年4月24日：
- 传来BSV下架币安的消息
- 币价从90美金一路狂跌到45美金
- 但成交量特别高，庄家在接住韭菜抛售的筹码

20天后：
- 币价突然一飞冲天，最高到了240美金
- **涨幅超过600%**

::: warning 透过现象看本质
所以，不要把项目、消息、币价划等号。听消息而入的，大概率买在高位。你看到的都是庄家想让你看到的，所谓一叶障目，正是指此。
:::

---

## 躺着挣钱的操作精髓

::: tip 总结
庄家的花招太多了，用尽一切办法逼迫你投降。

**你需要细心观察庄家是否完成了前四个动作，如果完成了，放心大胆的跟着买，怎么折腾也别下车。**

同时提醒：**不要用跌幅来判断底部**，底部之下还有底部，十八层地狱之下还有十八层地狱。
:::

在真正的底部建仓，然后就去休息了，啥也不看了。该读书读书，该工作工作，该抱老婆抱老婆，该生孩子生孩子。只要你跟上庄家的节奏，在庄家吃够币且币价低的时候介入，就可以躺着挣钱了。

**花钱让庄家为你打工，这叫富人思维。**
