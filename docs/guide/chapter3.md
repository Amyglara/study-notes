# 第三章：币市赚钱的真相

## 本章学习要点

::: tip 核心要义
1. **币圈盈利的真相是下跌。**
2. 跌幅可以取得低成本筹码与高倍数空间，庄家挣钱靠倍数，而非币价差价的绝对值。1元买10元卖，和0.1元买1元卖，没差别。
3. 币市的环境允许这么玩儿。
4. **不要幻想庄家会拉回原高位。** 拉回原高位对庄家来说不一定划算。
5. 不要关注自己的盈亏，关注庄家的盈亏，才能摆脱韭菜思维。
:::

---

## 一. 跌的越多，筹码越多

数字货币投资的本质是【做生意】，币价低的时候买入，币价高的时候卖出，中间的差价就是我们的盈利空间。读者如此、笔者如此、**庄家亦如此**。

所以，币价不下跌，庄家去哪里买便宜货呢？

股市也好、币市也好、外汇市场也好、期货市场也好，价格都是从下到上，从上到下，像波浪一样。往高深了说，这叫**周期**。往简单说，庄家故意打压币价，好在低位买货；买够了，才拉升币价，好在高位卖货，如是而已。

::: info 揭开障眼法
大家应当看到币价涨跌背后的真相，不要被所谓的项目、技术、前景、消息、监管这类障眼法迷惑。

- 如果庄家没囤够低价的币，项目由谁操盘，币价也不会拉伸。
- 如果庄家手里全是低价币，任何消息都阻止不了他拉伸币价。
- 如果庄家已经把货都在高位卖光了，任何人也阻止不了币价下跌。
:::

这就解释了为何：
- 区块链1.0的 BTC 可以从 20000 美金跌到 3000 美金，**跌幅 670%**
- 区块链2.0的 ETH 可以从 1422 美金跌到 81.93 美金，**跌幅 1735%**
- 区块链3.0的 EOS 可以从 23.08 美金跌到 1.52 美金，**跌幅 1515%**

> **利空是为了打压币价方便庄家低价吸收筹码，利好是为了拉升币价吸引韭菜来接盘。**

---

## 二. 跌的越多，拉伸空间越大

### 倍数比币价绝对值更重要

| 轮次 | 买入价（美金） | 卖出价（美金） | 利润率 |
| :--- | :--- | :--- | :--- |
| 第一轮 | 100 | 500 | **500%** |
| 第二轮 | 10 | 50 | **500%** |
| 第三轮 | 1 | 5 | **500%** |
| 第四轮 | 0.1 | 0.5 | **500%** |

庄家100元买入，拉到500元卖出，利润率500%。庄家10元买入，拉到50元卖出，利润率也是500%。

::: warning 重要结论
**大部分的币价不会再回到前高了**，因为拉回前高反而可能亏，或者利润率会降低。

以 EOS 为例：
- 2018年：从 3.8 美金涨到 23 美金，庄家利润率 600%
- 2019年：从 1.5 美金涨到 8.6 美金，庄家利润率 570%

币价虽然没有回到前高，但庄家的利润率是一样的。
:::

### 给韭菜解套不划算

假设币价从500跌到10元，这些价格区间已有大量被套韭菜。

- 从10元拉到50元，纯赚500%
- 拉到100就有风险了，庄家要真金白银给韭菜解套
- 被套的韭菜看到币价回来，大概率会卖掉，庄家要接盘

**拉回前高可能亏，不拉回前高一定赚，你是庄家如何抉择？**

---

## 三. 币圈规则允许这么玩儿

币市是可以通过打压币价的方式赚钱的，原因有四：

### 1. 币价可以无限小

股市中股票价格有下限（如0.01元人民币），但币圈不同。比特币可以细分到小数点后8位，很多山寨币的交易对价格可以显示为 `0.00000001`。

**数字货币价格的特殊性，给了庄家利用超级跌幅盈利的空间。**

### 2. 币币无监管

在股市，"**操纵证券罪**"是紧箍咒，高管、大股东等有限售制度。而币圈项目方可以随意抛售，上市即跌，联合交易所割韭菜的现象屡见不鲜。

### 3. 币市无成本

上市公司不会轻易让自己的股价暴跌几十倍，因为上市成本极高。但币圈项目上市成本很低，给交易所百万千万即可，因此项目方不珍惜，可以随意操纵币价。

### 4. 项目方没追求

小项目方没有大资本家的野心，只想赚几百几千万跑路，这个项目搞垮了，换下一个继续。

---

::: danger 总结
绝大多数币的币价都会一直往下走，走着走着就爆发一次，且永远回不了前高。

**紧盯币价是注定亏钱的。当你不关心自己的币价涨跌，开始关心庄家是否赚到钱了，你就不是韭菜了。**
:::
