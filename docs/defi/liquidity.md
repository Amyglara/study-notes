# 流动性挖矿

流动性挖矿（Liquidity Mining / Yield Farming）是 DeFi 最主要的被动收益方式——**给协议提供流动性，赚手续费 + 代币奖励**。

## 基本原理

```
你：提供 ETH + USDC（1:1 价值）进入流动性池
    ↓
DEX 用你的资金撮合其他用户的交易
    ↓
每笔交易收取手续费（Uniswap V3 约 0.05% / 0.3% / 1%）
    ↓
手续费按你的份额比例分给你
    ↓
协议可能额外发放代币（挖矿奖励）
```

## 无常损失（Impermanent Loss）—— LP 最大风险

::: danger 必须理解无常损失
这是很多新手亏钱的根源。
:::

**情景模拟**：

```
初始：你存入 1 ETH + $2000 USDC（ETH价格 $2000）
      你的份额 = 50% ETH + 50% USDC

ETH 涨到 $4000 后，套利者会把池子调成新比例：
      你的份额变成 0.707 ETH + $2828 USDC
      总价值 = $5656

如果你当初没做 LP，直接持有：
      1 ETH × $4000 + $2000 = $6000

无常损失 = $6000 - $5656 = $344（约 5.7%）
```

**规律**：
- 两个代币价格差距越大 → 无常损失越大
- 价格回到起点 → 无常损失消失（这就是"无常"的含义）
- 用两个相关性极强的代币做 LP（如 ETH/stETH）→ 无常损失极小

**何时做 LP 合适**：
- 手续费收入 > 无常损失 → 赚
- 市场横盘震荡 → LP 更合适
- 单边上涨行情 → 直接拿币更合适

## 实战：Uniswap V3 提供流动性

### 第一步：确定策略

```
选对：
  币对类型：稳定币对（USDC/USDT）最稳，波动对（ETH/USDC）次之，小币对高风险
  手续费档：
    0.01%  → 稳定币对（USDC/USDT）
    0.05%  → 主流大币（ETH/BTC）
    0.30%  → 一般代币
    1.00%  → 高波动 / 新上线代币
  价格区间：越窄 → 赚得越多（价格在区间内）= 也越容易失效
```

### 第二步：操作

1. 打开 [app.uniswap.org/pool](https://app.uniswap.org/pool)
2. "+ New Position"
3. 选择代币对 + 手续费档
4. 设置价格区间（保守：当前价格 ±30%）
5. 输入金额 → "Add" → 钱包签名

### 第三步：管理

- 定期检查价格是否还在区间内
- 超出区间 → 停止赚手续费 → 考虑重新设置
- 收取手续费：点击 "Collect fees"

## 其他挖矿方式

### Staking（质押）

最简单的被动收益，把代币锁定给协议：

| 项目 | 代币 | APY | 说明 |
|------|------|-----|------|
| Lido | stETH | ~3.5% | 以太坊质押，流动性质押 |
| Aave | 各代币 | 2-15% | 借贷协议存款 |
| PancakeSwap | CAKE | 5-20% | 单币质押 |
| Pendle | 多种 | 5-30% | 收益代币化协议 |

### 稳定币流动性（最低风险）

```
USDC/USDT 池：
  几乎零无常损失（两者都锚定$1）
  手续费 APY：0.5% - 5%（视流量）
  风险：智能合约风险、稳定币脱锚风险（USDC/USDT 极小）
```

**最成熟的稳定币协议**：
- **Curve Finance**：稳定币之王，TVL 数十亿
- **Convex**：在 Curve LP 的基础上提高收益

## 收益计算

```
年化收益率（APY）计算：

日手续费收入 / 你的本金 × 365 = APY

例：
  你的本金：$10,000
  昨日手续费收入：$8.22
  APY = $8.22 / $10,000 × 365 ≈ 30%

注意：APY 每天变化，高 APY 往往不持续
```

## 风险总结

| 风险类型 | 级别 | 说明 |
|---------|------|------|
| 无常损失 | ⭐⭐⭐ | 价格剧烈波动时损失本金 |
| 智能合约漏洞 | ⭐⭐⭐ | 协议被黑，本金归零 |
| Rug Pull | ⭐⭐⭐ | 新项目代币奖励高，团队跑路 |
| 价格超出区间 | ⭐⭐ | 停止赚手续费（不亏本金） |
| Gas 费 | ⭐ | 频繁操作消耗手续费 |

::: tip 新手建议
第一次做 LP，从 **USDC/USDT 稳定币对 + 知名协议（Curve/Uniswap）** 开始。  
本金不超过总资产的 10%。
:::
