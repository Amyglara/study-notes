# DeFi 概览

DeFi（Decentralized Finance，去中心化金融）是运行在区块链上的金融服务体系，无需银行、券商等中间机构。

## DeFi 核心协议

| 类型 | 代表项目 | 作用 |
|------|---------|------|
| DEX | Uniswap、Jupiter | 去中心化交易 |
| 借贷 | Aave、Compound | 抵押借贷 |
| 稳定币 | MakerDAO、Frax | 价值稳定 |
| 收益聚合 | Yearn、Convex | 自动化收益 |
| 衍生品 | dYdX、GMX | 合约交易 |

## TVL 是什么？

TVL（Total Value Locked，总锁仓量）是衡量 DeFi 协议规模的核心指标。  
TVL 越高 = 用户信任越高 = 协议越安全。

查看实时 TVL：[DefiLlama](https://defillama.com)

## 风险提示

::: warning 注意
DeFi 高收益背后有高风险：智能合约漏洞、流动性风险、价格操纵。  
新手入场先用小额试水。
:::
