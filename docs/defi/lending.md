# 借贷协议

DeFi 借贷让你可以**不卖币就拿到现金流**——用加密资产作抵押，借出稳定币使用。

## 为什么要抵押借贷？

```
场景：你持有 10 ETH，不想卖，但需要 $5000 周转

传统方式：卖掉 2.5 ETH（错过后续上涨，还要交税）

DeFi 方式：
  抵押 10 ETH → 借出 $8000 USDC（LTV 40%）
  用 $8000 做其他投资或支出
  ETH 不动，继续享受涨幅
  有钱时还款 + 利息，取回 ETH
```

## 核心概念

### LTV（贷款价值比）

```
LTV = 借款金额 / 抵押品价值

例：抵押 1 ETH（价值 $3000），最多借 $2100
    LTV = 2100 / 3000 = 70%

Aave 对 ETH 的最大 LTV = 82.5%（保守取 70% 更安全）
```

### 清算线（Liquidation Threshold）

当你的**健康因子（Health Factor）< 1** 时，会被清算：

```
健康因子 = 抵押品价值 × 清算阈值 / 借款金额

例：
  抵押 1 ETH = $3000，ETH清算阈值 85%
  借出 $2000 USDC
  健康因子 = 3000 × 0.85 / 2000 = 1.275  ← 安全

ETH 跌到 $2000：
  健康因子 = 2000 × 0.85 / 2000 = 0.85  ← 触发清算！
  清算人还掉你的债务，拿走你的抵押品（+5%奖励）
```

::: danger 清算警告
被清算不只是"平仓"——清算人会拿走超过你债务价值的抵押品作为奖励。  
建议：健康因子始终保持 > 2，极保守者 > 3。
:::

## 主流借贷协议

### Aave

```
网址：app.aave.com
支持链：以太坊 / Arbitrum / Base / Polygon / Optimism / Avalanche
代币：AAVE
```

**Aave 是 DeFi 借贷的标杆**，TVL 稳居前三：

- **存款赚利息**：存入 USDC 年化约 3-12%
- **抵押借款**：支持 ETH/BTC/MATIC/LINK 等主流资产
- **闪电贷**：同一笔交易内借还，无需抵押（套利/清算专用）
- **E-Mode**：同类资产（如 ETH/stETH）可提高 LTV

**Aave V3 新特性**：
- 跨链流动性
- 效率模式（E-Mode）
- 隔离模式（限制小币风险）

---

### Compound

```
网址：app.compound.finance
支持链：以太坊 / Base
代币：COMP
```

- DeFi 借贷的先驱
- 界面更简洁，适合新手
- Base 上的 Compound V3 Gas 费极低

---

### MakerDAO / Sky（DAI 发行方）

```
网址：app.spark.fi
支持链：以太坊
代币：MKR → SKY（品牌升级中）
```

- 不同于 Aave，Maker 通过抵押品**铸造 DAI（稳定币）**
- 你不是借别人的钱，而是铸造新 DAI
- 最低抵押率 150%（ETH）

---

### Venus（BNB Chain）

```
网址：app.venus.io
原链：BNB Chain
```

- BNB Chain 最大借贷协议
- 兼容 Compound 接口
- Gas 极低，适合小额借贷

## 实战操作：Aave 存款赚利息

1. 打开 [app.aave.com](https://app.aave.com)，连接钱包
2. 选择网络（Arbitrum 省 Gas）
3. 点击"Supply"选择资产（如 USDC）
4. 输入金额 → "Approve" → "Supply"
5. 获得 aToken（如 aUSDC），自动计息

**取回时**：直接"Withdraw"，aToken 销毁，本金 + 利息归还

## 进阶：杠杆循环（高风险！）

```
杠杆做多 ETH 操作流程：

1. 抵押 1 ETH（价值 $3000）
2. 借出 $2000 USDC（LTV 67%）
3. 用 $2000 USDC 买入 0.67 ETH
4. 再次抵押 0.67 ETH
5. 再借出 $1340 USDC ...
6. 重复，最终持有约 2-3 ETH 的多头敞口

ETH 涨 → 放大收益
ETH 跌 → 快速清算，损失本金
```

::: danger 杠杆操作风险极高
单次清算可能损失 30-50% 本金。  
仅建议已有丰富 DeFi 经验、有专门风控工具的用户操作。
:::

## 利率类型

| 类型 | 特点 | 适合 |
|------|------|------|
| 浮动利率 | 随资金池利用率变化 | 短期借款 |
| 固定利率（Aave V3） | 锁定利率，相对稳定 | 中长期规划 |

**利率曲线原理**：
```
资金池利用率 = 借出量 / 总存款量

利用率低（50%）→ 利率低（鼓励借款）
利用率高（90%）→ 利率急剧上升（鼓励存款 + 抑制借款）
```

## 借贷风险管理

| 建议 | 说明 |
|------|------|
| 健康因子 > 2 | 给价格波动留出缓冲空间 |
| 只借主流资产 | 避免借波动大的山寨币 |
| 设置价格提醒 | TradingView / Aave 原生提醒 |
| 不超仓 | 借出资金不超过总资产 30% |
| 关注利率变化 | 高利用率时利率会飙升 |
