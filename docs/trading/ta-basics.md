# 技术分析入门

技术分析（TA）是通过**历史价格和成交量**来预判未来走势的方法。不是算命，是概率游戏。

## K 线基础

### 一根 K 线的解读

```
      │  ← 上影线（最高价 - 实体顶）
  ┌───┤
  │   │  ← 实体（绿/红）
  │   │    绿柱：收盘 > 开盘（上涨）
  └───┤    红柱：收盘 < 开盘（下跌）
      │  ← 下影线（实体底 - 最低价）

一根K线包含：开盘价 / 收盘价 / 最高价 / 最低价
```

### 常见 K 线形态

| 形态 | 外观 | 含义 |
|------|------|------|
| 大阳线 | 长绿实体，短影线 | 强烈看多，买盘主导 |
| 大阴线 | 长红实体，短影线 | 强烈看空，卖盘主导 |
| 十字星 | 几乎无实体 | 多空僵持，方向不明 |
| 锤头线 | 短实体 + 长下影线 | 底部信号（下跌被承接） |
| 射击之星 | 短实体 + 长上影线 | 顶部信号（上涨被压制） |
| 吞没形态 | 大阳线吞没前一阴线 | 看多反转信号 |

## 趋势线

### 上升趋势

```
价格走势：
    /\/\/\/  → 每次高点比上一个高，每次低点也比上一个高
  
关键线：支撑线（连接低点）
操作：回踩支撑不破 → 买入机会
```

### 下降趋势

```
价格走势：
    \/\/\/  → 每次高点比上一个低，每次低点也比上一个低
  
关键线：压力线（连接高点）
操作：反弹到压力不破 → 卖出机会
```

### 趋势线画法
- 连接至少 **2 个有效高点或低点**（3个更可信）
- 被强势突破（收盘价穿越）才算有效突破
- 反弹回测突破线，通常是好的入场机会

## 支撑与阻力

```
阻力位（Resistance）：价格上涨时遇到的"天花板"
支撑位（Support）：价格下跌时遇到的"地板"

重要规律：
  阻力被突破后 → 变成支撑
  支撑被跌破后 → 变成阻力

  这叫"支撑阻力反转"，极为重要
```

### 关键价格位
- **整数关口**：$50,000 / $100,000 / $3000 等
- **历史高低点**：前期高点/低点往往成为关键位
- **成交量密集区**：大量筹码换手的价格区间
- **移动均线**：MA20 / MA50 / MA200 常作为动态支撑

## 移动平均线（MA）

```
MA20 = 过去 20 根K线收盘价的平均值

意义：
  价格在 MA20 上方 → 短期偏强
  价格在 MA200 上方 → 长期偏强
  MA20 上穿 MA200（金叉）→ 看多信号
  MA20 下穿 MA200（死叉）→ 看空信号
```

**EMA（指数移动平均）**：对近期价格给予更高权重，反应更灵敏，更常用。

| 均线 | 周期 | 用途 |
|------|------|------|
| EMA9 | 极短 | 日内快线 |
| EMA20 | 短期 | 日线主要参考 |
| EMA50 | 中期 | 牛熊分界参考 |
| EMA200 | 长期 | 牛市/熊市判断基准 |

## 成交量分析

```
价格上涨 + 成交量放大 → 买盘真实，可持续
价格上涨 + 成交量萎缩 → 买盘衰竭，谨慎追高

价格下跌 + 成交量放大 → 卖盘恐慌，可能接近底部
价格下跌 + 成交量萎缩 → 惯性下跌，等待止跌信号
```

## 常见形态

### 双顶 / 双底（W底 / M顶）

```
双底（W形）：               双顶（M形）：
        _                   _     _
   _   / \   _            /  \  /  \
  / \ /   \_/ \          /    \/    \
 /   V         \        /            \
← 看多，突破颈线入场 →   ← 看空，跌破颈线入场 →
```

### 头肩顶 / 头肩底

```
头肩顶（反转看空）：

     /\      ← 头（最高点）
    /  \
/\/    \/\  ← 两肩（高度相近）
            颈线（两肩的低点连线）
---颈线---
跌破颈线 → 卖出信号
```

## 时间周期选择

| 周期 | 适合 | 信号有效期 |
|------|------|---------|
| 1分钟 / 5分钟 | 高频日内 | 几分钟 |
| 1小时 / 4小时 | 短线波段 | 几小时到几天 |
| 日线 | 主流波段交易 | 数天到数周 |
| 周线 | 中长期布局 | 数周到数月 |

> 💡 **多周期共振**：日线方向确认 + 4H入场信号，成功率更高
