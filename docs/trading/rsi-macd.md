# RSI + MACD 共振信号

这是麒麟会扫描器的核心策略——**三指标共振过滤**，减少假信号，提高胜率。

## 三个核心指标

### RSI（相对强弱指数）

```
RSI 范围：0 - 100

70 以上：超买区间（价格可能过热）
30 以下：超卖区间（价格可能超跌）
50 附近：中性区间

计算方式：
  RSI = 100 - 100 / (1 + RS)
  RS = N日平均上涨幅度 / N日平均下跌幅度
  默认周期：N = 14
```

**RSI 使用技巧**：

| 信号 | 含义 |
|------|------|
| RSI 从超卖（<30）回升穿越 30 | 看多做多信号 |
| RSI 从超买（>70）回落穿越 70 | 看空做空信号 |
| RSI 底背离（价格新低，RSI 未新低） | 看多反转信号 |
| RSI 顶背离（价格新高，RSI 未新高） | 看空反转信号 |
| RSI 保持 50 以上 | 强势行情（多头市场） |

---

### MACD（指数平滑移动平均线）

```
MACD 由三部分组成：
  DIF（快线）= EMA12 - EMA26
  DEA（慢线）= DIF 的 9日EMA
  MACD柱 = (DIF - DEA) × 2

信号解读：
  DIF 上穿 DEA（金叉）→ 看多
  DIF 下穿 DEA（死叉）→ 看空
  柱由负转正 → 动能转向看多
  柱由正转负 → 动能转向看空
  背离 → 同 RSI 背离逻辑
```

**MACD 背离**：
```
顶背离：
  价格创新高，但 MACD 柱没有创新高
  → 上涨动能减弱，注意反转

底背离：
  价格创新低，但 MACD 柱没有创新低
  → 下跌动能减弱，关注做多机会
```

---

### EMA（指数移动平均线）

在共振策略中，用 EMA 判断**趋势方向**：

```
多头排列（看多做多前提）：
  EMA9 > EMA20 > EMA50

空头排列（看空做空前提）：
  EMA9 < EMA20 < EMA50

价格站上 EMA200 = 长期牛市确认
价格跌破 EMA200 = 长期熊市警惕
```

## 三指标共振——入场逻辑

### 做多信号（三条全满足）

```
✅ EMA：EMA9 > EMA20（短期看多）
✅ RSI：从超卖区（30-45区间）向上
✅ MACD：DIF 上穿 DEA 金叉，或柱由负转正

入场条件同时满足 → 开多
```

### 做空信号（三条全满足）

```
✅ EMA：EMA9 < EMA20（短期看空）
✅ RSI：从超买区（55-70区间）向下
✅ MACD：DIF 下穿 DEA 死叉，或柱由正转负

入场条件同时满足 → 开空
```

## 止损与止盈

::: warning 止损是保命线，不设止损 = 裸奔
每笔交易必须在入场前确定止损价，不能靠感觉。
:::

### 止损设置原则

```
做多止损位：
  方法1：最近的支撑位下方（约 1-3%）
  方法2：EMA50 下方
  方法3：入场价 × (1 - 最大容忍亏损%)

做空止损位：
  方法1：最近的阻力位上方（约 1-3%）
  方法2：EMA50 上方
  方法3：入场价 × (1 + 最大容忍亏损%)
```

### 三档止盈（TP1 / TP2 / TP3）

麒麟会扫描器标配三档止盈：

```
TP1（第一目标）：风险收益比 1:1.5
  小仓位兑现，保护利润

TP2（第二目标）：风险收益比 1:2.5
  主仓兑现，收益落袋

TP3（最大目标）：风险收益比 1:4+
  留余仓博趋势行情

例：
  入场价：$100
  止损位：$95（-5%）
  TP1：$107.5（+7.5%）  风险收益比 1:1.5
  TP2：$112.5（+12.5%） 风险收益比 1:2.5
  TP3：$120  （+20%）   风险收益比 1:4
```

## 实战案例

### 案例：BTC 4H 级别做多信号

```
2024年某日，BTC 4H图：

价格：$62,000
EMA9（$62,200）> EMA20（$61,800）✅  — 短期多头排列
RSI：从 38 回升至 52 ✅  — 超卖回暖
MACD：DIF 刚完成金叉 ✅  — 动能转多

三指标共振 → 触发做多信号

入场：$62,200（金叉确认后开盘价）
止损：$60,500（近期低点下方 0.5%）
TP1：$64,750（+4%）
TP2：$66,450（+6.8%）
TP3：$70,000（+12.5%）

风险：入场价 - 止损 = $1,700（约 2.7% 亏损）
```

## 提升胜率的过滤条件

| 过滤器 | 方法 |
|--------|------|
| 多周期验证 | 日线方向与 4H 一致，胜率更高 |
| 成交量确认 | 信号出现时成交量同步放大 |
| 市场情绪 | 恐慌贪婪指数极恐（<20）+ 做多信号，反弹概率更高 |
| 基本面过滤 | 避开重大消息前（FOMC/ETF数据等）入场 |
| 排除稳定币 | 确保扫描对象是真实的波动标的 |
