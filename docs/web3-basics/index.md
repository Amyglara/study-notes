# 什么是 Web3

Web3 是互联网发展的第三个阶段，核心思想是**去中心化**——数据和资产的控制权回归用户本人，而不是平台公司。

## Web1 / Web2 / Web3 对比

| 阶段 | 时代 | 特征 | 典型代表 |
|------|------|------|---------|
| Web1 | 1990s-2000s | 只读，静态页面 | 早期门户网站 |
| Web2 | 2000s-2020s | 读写，平台控制数据 | 微信、微博、抖音 |
| Web3 | 2020s- | 读写拥有，用户控制 | 以太坊、Uniswap |

## Web3 的核心组成

### 1. 区块链
去中心化的账本，记录所有交易，不可篡改。

### 2. 智能合约
运行在区块链上的程序，自动执行，无需信任第三方。

### 3. 钱包
你在 Web3 世界的"账号"，私钥即身份。

### 4. 代币 (Token)
Web3 世界的价值载体，分为同质化代币（ERC-20）和非同质化代币（NFT）。

## 为什么要学 Web3？

> 现在入场学习，就像 2000 年学互联网一样。

- 💰 **财富机会**：早期认知红利
