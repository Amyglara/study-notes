# 钱包与私钥

Web3 里没有"账号注册"——你的钱包**就是你的身份**，私钥丢了，资产永久消失。

## 钱包的本质

> 钱包不"存"币，币永远在区块链上。钱包存的是你的**私钥**——证明你有权转移那些币的密码。

```
助记词（12/24个英文单词）
    ↓ 生成
私钥（256位随机数）
    ↓ 推导
公钥（私钥经椭圆曲线算法推导）
    ↓ 哈希
地址（0x...，对外公开的收款地址）
```

这个推导是**单向的**：地址无法反推出私钥。

## 助记词 vs 私钥

| 对比 | 助记词 | 私钥 |
|------|-------|------|
| 形式 | 12/24个英文单词 | 64位16进制字符串 |
| 作用 | 恢复整个钱包（含所有账户） | 控制单个账户 |
| 备份优先级 | ⭐⭐⭐ 首选 | 次选 |
| 示例 | `witch collapse practice feed...` | `0x4c0883a69102937d623...` |

::: danger 铁律：助记词/私钥绝不能上网
- 不截图存手机相册
- 不发微信/邮件/云笔记
- 不输入任何不明网站
- **纸抄 + 离线保存**，多地备份
:::

## 钱包类型

### 热钱包（联网）
日常使用，方便但有被攻击风险。

| 钱包 | 支持链 | 特点 |
|------|-------|------|
| **MetaMask** | EVM 链（ETH/BSC/Polygon等） | 最流行，浏览器插件 |
| **Rabby** | EVM 链 | 交易风险提示更强，推荐替代MM |
| **Phantom** | Solana / ETH | Solana 最流行 |
| **OKX Web3 钱包** | 多链 | 国内用户友好 |
| **Trust Wallet** | 多链 | 手机端首选 |

### 冷钱包（离线）
大额资产必备，私钥永不联网。

| 品牌 | 价格 | 推荐人群 |
|------|------|---------|
| **Ledger** | ¥600-1200 | 主流选择 |
| **Trezor** | ¥500-1500 | 开源党首选 |
| **OneKey** | ¥400-800 | 国产，中文友好 |

> 💡 **经验**：持有 5 万以上的资产，冷钱包是必须的，不是可选的。

## MetaMask 快速上手

### 1. 安装
- 官网：[metamask.io](https://metamask.io)（注意辨别钓鱼网站）
- 只从 Chrome Web Store 安装

### 2. 创建钱包
1. 点击"创建新钱包"
2. 设置密码（只保护本地，非私钥）
3. **抄写 12 个助记词**（顺序不能错）
4. 验证助记词

### 3. 添加网络
MetaMask 默认只有以太坊主网，手动添加其他链：

```
BNB Chain:
  网络名称: BNB Smart Chain
  RPC URL: https://bsc-dataseed.binance.org/
  Chain ID: 56
  代币符号: BNB
  区块浏览器: https://bscscan.com
```

或直接访问 [chainlist.org](https://chainlist.org) 一键添加。

## 常见攻击手法（防骗必读）

### 1. 钓鱼网站
- 伪造 Uniswap、MetaMask 等官网
- 防御：收藏官网链接，不点搜索引擎广告

### 2. 假空投
- "你的地址有 $1000 代币待领取，点击授权"
- 本质：让你签授权，然后转走所有资产
- 防御：不明空投一律不动

### 3. 无限授权
- 你调用 DEX 时会"批准"合约使用你的代币
- 部分恶意合约设置无限额度
- 防御：定期用 [revoke.cash](https://revoke.cash) 检查并撤销多余授权

### 4. 私信骗局
- Discord/Telegram 里"客服"主动找你，要求你输入助记词"验证"
- **没有任何正规项目需要你的助记词**
